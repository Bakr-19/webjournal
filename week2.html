Week 2 CSS & HTML

What We Cover On This Week?

Week 2 or CSS and HTML Week. This week we learnt the fundamentals of CSS and HTML. We learnt things like tag, different selectors, attributes, fonts …etc CSS is a declarative language and HTML is a Markup Language which means they are not your normal coding languages, rather they are used to create websites and website design. In fact im writing this text in HTML.

How did i find this week?

Prior to this course covering CSS and HTML, I had about 2 hours of experience. Not much, but nonetheless enough to get your head around what these languages do. In a nutshell, i found this week to very easy because like i said these two languages are not your normal coding languages, they are declarative languages that tells the browser what content should go where and the like of.

What kind of tips can you give us?

I think using CSS and HTML for only a couple of days, i CANT call myself a professional of any sort but for people that are looking to get into CSS and HTML, then remember it is not complicated. Here is a few tips, although i might change my mind on them:

  1. Use W3Schools insead of MDN, because its more dumbed down
  2. Use youtube and possible MDN if W3S is not giving you answers
  3. Take inspirations from other websites
  4. Remember CSS and HTML ONLY are not going to make website looks the “Professional” you need Javascript for that one
